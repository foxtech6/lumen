# Requirments
- docker (latest)
- docker-compose (latest)

## Setup
Build and up project:
```sh
make all
```
Only build:
```sh
make build
```
Only up:
```sh
make up
```
Down project:
```sh
make down
```
Run tests:
```sh
make test
```
#

When the project has been successfully setup, then you can go to the address in the browser and test it:

```
http://localhost:8089
```
